# IKM Road Runner LLC Website

A mobile-first marketing site for IKM Road Runner LLC built with React, Vite, React Router, Tailwind CSS, and Helmet for page-level SEO tags.

## Pages
- Home (hero with USDOT 4486692 and MC 1772922-C)
- About
- Services
- Service Area
- Contact (mailto quote form)

## Development
1. Install dependencies (ensure your network/proxy allows npm registry access):
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

> Note: React Router handles routing; each page includes unique `<Helmet>` metadata for SEO.
